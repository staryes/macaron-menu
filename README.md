# Macaron Menu

這是一個賣幼童科學玩具的網站，旨在提供有趣且富有教育意義的科學玩具，讓幼童在遊戲中學習科學知識，啟發探索與創造力。

## 專案簡介
本網站展示各式科學玩具，並提供品牌故事、產品介紹等資訊，協助家長選購適合幼童的科學玩具。

## 技術棧（Tech Stack）
- [Nuxt 3](https://nuxt.com/)：基於 Vue 3 的現代前端框架
- [Vue 3](https://vuejs.org/)：漸進式 JavaScript 框架
- [TypeScript](https://www.typescriptlang.org/)：型別安全的 JavaScript 超集
- [Vite](https://vitejs.dev/)：快速的前端建構工具
- [CSS](https://developer.mozilla.org/zh-TW/docs/Web/CSS)：樣式設計

## 專案結構
- `app.vue`：主應用入口
- `nuxt.config.ts`：Nuxt 設定檔
- `assets/`：靜態資源（如 CSS）
- `components/`：可重用元件
- `pages/`：頁面檔案
- `public/`：公開靜態檔案

## 如何啟動
1. 安裝依賴：`npm install`
2. 啟動開發伺服器：`npm run dev`
